# Kozmik Rehber (Cosmic Guide) 🌌

Kullanıcıların doğum haritası, günlük burç yorumları ve tarot kartı çekimi gibi astrolojik deneyimlere ulaşabildiği, modern ve estetik bir web uygulamasıdır.

## 🛠️ Teknolojiler
* **Frontend:** HTML5, CSS3 (Modern Flexbox/Grid yapısı)
* **İnteraktivite:** JavaScript (DOM Manipülasyonu)
* **Tasarım:** Estetik ve mistik bir kullanıcı arayüzü (UI)

## ✨ Temel Özellikler
* **Burç Rehberi:** Günlük burç özelliklerinin dinamik görüntülenmesi.
* **Tarot Kartı:** Rastgele kart çekme ve anlamlarını listeleme.
* **Kozmik Tasarım:** Koyu tema ve gökyüzü temalı görsel öğeler.
* **Kullanıcı Etkileşimi:** JavaScript ile sağlanan dinamik içerik güncellemeleri.



## 🏗️ Mimari Yapı
Proje tamamen istemci taraflı (client-side) çalışan, hız odaklı bir yapıya sahiptir:
1. **HTML:** Uygulamanın iskeleti ve içerik yapısı.
2. **CSS:** Mistik bir atmosfer yaratan animasyonlar ve stil tanımları.
3. **JS:** Kullanıcı etkileşimlerini yöneten, burç ve tarot verilerini işleyen mantık katmanı.

## 🚀 Kurulum

Projeyi kendi bilgisayarınızda geliştirmek veya çalıştırmak için aşağıdaki adımları izleyebilirsiniz:

**1. Proje Dosyalarını İndirin:** Terminalinizi açın ve kodları bilgisayarınıza çekmek için şu komutu çalıştırın:
```bash
git clone https://github.com/ikbaltorun/kozmik-rehber.git
```
**2. Projeyi Çalıştırın:** İndirdiğiniz klasörün içindeki index.html dosyasını herhangi bir web tarayıcısında (Chrome, Safari, Edge vb.) çift tıklayarak açmanız yeterlidir. (Geliştiriciler, VS Code üzerinden "Live Server" eklentisiyle de anlık önizleme yapabilirler).
